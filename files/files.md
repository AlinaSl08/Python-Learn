✅ **Файл** (англ. file) — именованная область данных на носителе информации.

✅ **Файл ввода** — файл, из которого программа читает данные.

✅ **Файл вывода** — файл, куда сохраняются данные.

✅ Когда в программе используется файл, как правило требуется выполнить три шага:

+ **Открыть файл.** В процессе открытия файла создается связь между файлом и программой. Открытие файла вывода обычно создает файл на диске и позволяет программе записать в него данные. Открытие файла ввода позволяет программе прочитать данные из файла.
+ **Обработать файл.** На этом шаге данные либо записываются в файл (если это файл вывода), либо считываются из файла (если это файл ввода).
+ **Закрыть файл.** После использования файла программой **его нужно закрыть**, тем самым освободить ресурс и разорвать связь файла с программой.

✅ Существует два типа файлов:

+ **Текстовый файл.** Содержит данные, которые были закодированы в виде текста при помощи такой схемы кодирования, как ASCII или Юникод.
+ **Двоичный (бинарный) файл.** Содержит данные, которые не были преобразованы в текст. Данные, которые помещены в двоичный файл, предназначены только для чтения программой, и такой файл невозможно просмотреть в текстовом редакторе.

✅ Способы получения доступа к данным в файле:

+ **Последовательный.** Выдает порции информации одну за другой. Не позволяет переходить сразу к нужной части данных, сначала придется прочитать все предыдущие.
+ **Прямой или произвольный.** Позволяет переходить непосредственно к любой порции данных, не читая предыдущие.

✅ **Имя файла**, часто включает **расширение** (например, `.txt`, `.jpg`, `.py`), определяющее тип содержимого.

✅ **Путь файла (или путь к файлу)** — последовательное указание имен папок, через которые надо пройти, чтобы добраться до объекта. Пути к файлу бывают двух типов:

+ **Абсолютный путь** – полный путь к файлу, показывающий **точное место** его расположения. Он всегда один и тот же, пока файл не перемещен.
+ **Относительный путь** – привязан к какой-либо "отправной точке" и указан по отношению к ней.
